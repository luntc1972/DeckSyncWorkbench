Install The Optional DeckFlow Moxfield Extension

Browsers do not allow this site to install the extension automatically. You need to load it yourself, then return to DeckFlow and retry the import.

Current status

The extension is not published to the Chrome Web Store yet, so the current install path is Load unpacked.

The DeckFlow bridge is intentionally limited to localhost and 127.0.0.1 so the extension does not inject into unrelated sites.

Install steps

  1. Open the repository or downloaded project files on your machine.
  2. Open browser-extensions/moxfield-tag-exporter.
  3. In Chrome or Edge, open chrome://extensions.
  4. Turn on Developer mode.
  5. Click Load unpacked.
  6. Select the browser-extensions/moxfield-tag-exporter folder.
  7. Return to DeckFlow and submit the Moxfield URL again.

What it does

The extension fetches the Moxfield deck in the browser, converts it to Moxfield bulk-edit text, and passes that text back to DeckFlow so the normal import flow can continue.

If you self-host DeckFlow on a different origin, update the extension manifest's bridge matches before loading it.
