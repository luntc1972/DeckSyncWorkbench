The DeckFlow bridge now uses an allowed-origins list in the extension settings. Add each DeckFlow origin you trust there before using browser-side Moxfield imports.

Install steps

  1. Click Download above to save deckflow-bridge.zip.
  2. Unzip it anywhere on your machine.
  3. In Chrome or Edge, open chrome://extensions.
  4. Turn on Developer mode.
  5. Click Load unpacked.
  6. Select the unzipped deckflow-bridge folder.
  7. Click the extension's details page, open Extension options, and add this DeckFlow origin to the allow list.
  8. Return to DeckFlow and submit the Moxfield URL again.

What it does

The DeckFlow Bridge fetches the Moxfield deck in your logged-in browser session when the DeckFlow server cannot reach it directly, then hands the deck data back to DeckFlow so the normal import flow can continue.

The bridge injects broadly, but it only responds to origins you explicitly allow in extension options.
